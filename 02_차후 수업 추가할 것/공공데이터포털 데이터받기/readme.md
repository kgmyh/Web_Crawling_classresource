  - 지금까지 신청한 내역이 보이고 선택해서 `개발계정 상세`로 들어간다.
- `개발계정 상세` 에서 인증키를 복사한다.
- 데이터 상세 페이지(개발계정 상세 페이지에서는 제목 옆 상세설명 버튼 클릭)로 이동 후 요청 방식을 확인한다.
